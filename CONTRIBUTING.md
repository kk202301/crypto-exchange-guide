# 贡献指南 / Contributing Guide

本项目持续收录主流中心化交易所（CEX）的注册流程、KYC认证要求、
安全配置建议及常见问题解答，面向新手用户及海外华人加密货币用户群体。

## 你可以贡献什么

- 补充或更新某个交易所的注册流程（平台界面更新后步骤可能变化）
- 新增某个交易所的 KYC 常见问题
- 修正文档中过时的信息或错误描述
- 补充特定地区用户（如马来西亚、泰国、台湾等）的使用差异说明
- 翻译现有内容为其他语言版本

## 贡献方式

1. Fork 本仓库
2. 在对应交易所目录下修改或新增 `README.md` 内容
3. 提交 Pull Request，并在 PR 描述中说明更新内容及依据来源
4. 维护者审核后合并

## 内容规范

- 描述操作步骤时，以用户实际看到的界面文字为准
- 涉及费率、限额等数据，请注明信息来源及查阅时间
- 不接受纯推广性质的内容，所有注册链接须在文档顶部以透明方式声明

## 推广链接声明

本项目部分注册链接为维护者的推广链接。
用户通过这些链接注册后，双方均可获得平台提供的手续费优惠。
所有链接均指向交易所官方域名，不涉及第三方跳转或钓鱼风险。

## 提交 Issue

如果你在注册或使用过程中遇到本文档未覆盖的问题，欢迎通过
[Issues](../../issues) 提交，我们会及时补充相关内容。

## 免责声明

本项目仅提供操作流程说明，不构成任何投资建议。
加密货币交易存在市场风险，请用户自行评估后决策。
