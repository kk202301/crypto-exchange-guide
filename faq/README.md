# 通用常见问题解答

## 注册类

**Q：验证码收不到怎么办？**
检查垃圾邮件文件夹；确认邮箱地址拼写正确；验证码通常有效期10至30分钟，
过期后重新获取；部分邮件服务商有延迟，等待1至2分钟后再查收。

**Q：注册时提示IP受限怎么办？**
切换网络环境后重试。部分地区的网络出口IP可能在交易所的限制名单内，
更换网络后通常可以正常注册。

**Q：同一手机号可以注册多个账户吗？**
不可以。主流交易所均限制一个手机号对应一个账户，邮箱同理。
如需多账户请使用不同联系方式注册。

## KYC认证类

**Q：KYC认证需要多久？**
通常数分钟至24小时，高峰期可能延长至48小时。
审核期间账户可以登录，但充值、交易、提现功能受限。

**Q：人脸识别一直失败怎么办？**
确保正面朝向摄像头、光线充足、无遮挡物（眼镜、口罩等）；
避免逆光环境；使用设备前置摄像头而非截图或照片代替。

**Q：中国护照可以用于KYC认证吗？**
可以。币安、欧易、Bybit均接受中国护照及港澳台居民来往内地通行证
等有效证件完成KYC认证。

## 安全设置类

**Q：什么是2FA双重验证，为什么必须开启？**
2FA（双重验证）是在密码之外增加的第二层身份验证，每次登录或提现时
需要额外输入动态验证码。即使密码泄露，攻击者也无法在没有2FA设备的
情况下访问账户。推荐使用Google Authenticator而非短信验证，
可有效防范SIM卡劫持攻击。

**Q：Google Authenticator设备丢失怎么办？**
注册2FA时务必保存平台提供的备份密钥（Recovery Key），
妥善存储在密码管理器或离线位置。设备丢失后凭备份密钥可在新设备恢复。

## 入金类

**Q：P2P和C2C入金安全吗？**
安全，前提是通过交易所官方平台内的P2P/C2C功能操作。
平台会对资金进行托管，买方付款后卖方才能放币，
不要在平台外与陌生人直接转账。

**Q：新手第一次入金建议用哪种方式？**
建议使用P2P/C2C通道，支持支付宝、微信等熟悉的支付方式，
操作门槛低，无需绑定银行卡到境外平台，是华语用户最常用的起点。
