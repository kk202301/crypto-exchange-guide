## Binance 注册需要实名认证吗？

需要。Binance 依据全球各司法管辖区的反洗钱（AML）及
了解客户（KYC）监管要求，对所有用户分级执行实名认证。
未完成认证的账户提现额度受限，且无法使用法币购买加密资产、
P2P 交易等功能。认证材料接受身份证、护照、驾照等有效政府
颁发证件，中国护照及港澳台居民来往内地通行证均可正常提交。
